babushka-deps
=============

Personal babushka dependencies


## Getting Started

### Preparation

Create and change ownership of `/usr/local`

```
mkdir -p /usr/local
sudo chown -R `whoami` /usr/local
```

### Installing

Install Babushka

```
sh -c "`curl https://babushka.me/up`"
```

### Using Deps

```
babushka v2
```
